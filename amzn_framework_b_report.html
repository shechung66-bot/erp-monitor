一句话投资逻辑

Amazon 正在用整个公司的经营现金流(TTM $148.5B)押注 AI 基建。AWS 重新加速到 15 个季度最快(+28%),广告业务跑率破 $700 亿/年,利润率创历史新高 — 但 $200B 的 2026 资本开支让自由现金流逼近于零。这是一道"现在投入、未来兑现"的 ROIC 信仰题。

模块一

公司基本概览与核心业务

Amazon 早已不是一家"网购公司"。它是一个由零售飞轮供能、由 AWS 与广告两台高毛利引擎驱动的复合体 — 服务收入(AWS+第三方+广告+订阅)在 FY2025 已占总收入约 59%。

模块二

Quick Scan 的口号是"现金为王"。Amazon 在利润表上交出史上最强季度,但在现金流量表上,$200B 的 Capex 计划正在把自由现金流抽干 — 这是理解这家公司当前估值的唯一钥匙。

模块四

盈利能力与商业模式

核心问题:这门生意是否"越做越赚钱"?答案是肯定的 —— 但驱动力来自业务组合的结构性迁移,而非单一业务的提价。

Amazon 的飞轮是:零售规模 → Prime 会员黏性 → 第三方卖家与广告 → AWS 现金牛再投资。过去几年,收入结构正从"低毛利商品零售"持续向"高毛利服务"迁移 —— AWS 经营利润率约 37.8%、广告带软件级增量利润率。每多一个百分点的广告/AWS 增长,对 EPS 的贡献都远大于等量的零售增长。这正是混合经营利润率能创新高(13.1%)的根本原因。

单位经济:北美零售经营利润率升至 9.0%,说明随着物流密度与自动化提升,单均履约成本在下降,零售本身的 unit economics 正在改善;AWS 与广告则天然具备高 LTV / 低边际成本特征。综合看,这是一门规模越大、利润率越高的生意。

模块五

护城河与五力模型

Amazon 拥有当今科技界最宽、最多层次的护城河之一:网络效应、规模成本优势、切换成本与垂直整合并存。但云端的竞争强度不容忽视。

新进入者威胁低

云与物流均需千亿级资本与多年规模积累,几乎不可复制。$200B Capex 本身就是一道护城河。

替代品威胁中

云有 Azure/GCP 替代,零售有 Walmart/Temu;但 Prime 与 AWS 的切换成本显著抬高了替代门槛。

买方议价能力低–中

消费者高度分散,议价力弱;大型 AWS 企业客户有一定议价力,但被长期合约与迁移成本锁定。

供方议价能力低

Amazon 是多数供应商的最大买家;自研 Trainium 进一步削弱对 NVDA 的议价依赖。

现有竞争者竞争强度高

这是五力中唯一的高压项:云端 Azure(+40%)、Google Cloud(+63%)增速更快;零售面对 Walmart、Temu、Shein;广告面对 Google、Meta。竞争不会摧毁护城河,但会持续压制份额扩张速度与定价空间。

护城河类型:(1)网络效应 —— 卖家与买家互相吸引的双边市场;(2)规模成本优势 —— 全球最大的物流与数据中心网络;(3)切换成本 —— Prime 生态与 AWS 技术栈深度绑定;(4)垂直整合 —— 自研芯片把成本与性能掌握在自己手中。综合护城河评级:宽且持久,但需为"云份额增速被追赶"打折扣。

模块六

管理层与治理评估

执行力与兑现能力:CEO Andrew Jassy 上任后,在成本纪律(物流网络重构、裁员)与 AWS 重新加速上交出了成绩;Q1'26 收入与利润均显著超指引(EPS $2.78 vs 预期 $1.63),历史上对季度指引的兑现度总体可靠。

资本配置哲学:这是评估管理层最关键的一点。Amazon 选择把几乎全部经营现金流再投资到 Capex 与 AI,而不是回购或分红。这与当年建设物流与 AWS 的逻辑一脉相承 —— 区别只在于绝对金额的量级($200B)。这是一种"延迟满足"式的配置,赌的是未来 ROIC。对长期股东友好,但对追求短期现金回报的投资者不友好。

内部交易与治理:创始人 Bezos 的减持多为预设的 10b5-1 计划(常规分散化与慈善/Blue Origin 资金需求),不宜过度解读为信心信号。董事会结构成熟、信息披露透明度高。治理风险整体偏低。

治理判断

管理层与长期股东利益基本一致,执行力强、配置纪律清晰。唯一需要持续审视的是:$200B 量级的资本配置一旦 ROIC 不及预期,纠错的代价会非常高 —— 这是"信任管理层"与"独立验证"必须并存的地方。

核心投资逻辑(三句话)

1. 增长靠什么? AWS 重新加速(15 季最快 +28%)+ 广告破 $700 亿跑率 + 零售利润率结构性提升,三台引擎同时点火,backlog $364B 提供多年可见度。

2. 空间有多大? 云/AI 基建($2T+ TAM)、数字广告($1T+)、全球零售低渗透,加上自研芯片与 Kuiper 期权,长期天花板足够高。

3. 钱能不能赚? 这是唯一的问号。经营层面极能赚现金(OCF $148.5B),但全部被 $200B Capex 吃掉,FCF 接近零。能不能赚"自由现金流",取决于这轮 AI 投资的 ROIC 兑现 —— 这是一道 2027–28 才能验证的信仰题。

跨框架对照:为什么 A 框架说"等"、B 框架说"持有"?
估值差距取证(Valuation Gap Forensics)

在七股报告(Framework A,FCF 锚定)中,AMZN 被列为 Tier 3·等待,FY2029 概率加权目标 ~$253、IRR -1.5% —— 因为 A 框架以"每股 FCF × P/FCF"为锚,而 AMZN 的 FCF 已被 Capex 抽干到接近零,锚点本身近乎失效。本次 Framework B 用利润/EV/相对估值法,得出概率加权 ~$315、约 +19% 上行,更建设性。两个框架 ~$60 的差距,几乎 100% 来自同一个变量:自由现金流。A 框架惩罚"当下 FCF 为零",B 框架奖励"被再投资的盈利能力"。谁对,取决于 Capex 的 ROIC 能否兑现 —— 若 2027+ FCF 强劲转正,B 对;若 Capex 沦为价值陷阱,A 对。这正是 AMZN 必须用 B 框架(而非 FCF 锚定的 A 框架)估值的根本原因。
